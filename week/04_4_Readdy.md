# 版本
## v1

- 從世界觀 ~ 情節設定的 MVP
- 提供下載專案功能

## v1.2

- 完善故事編輯功能的 MVP

## v1.3

1. 深/淺色介面模式
   - 側邊欄底部多了「切換深色模式 / 切換淺色模式」按鈕（月亮/太陽 icon）
   - 切換偏好會自動儲存，下次開啟也會記住
   - 全站 8 個頁面 + 所有共用元件 都已完整支援深色模式，石墨黑底搭配柔和文字顏色

2. 角色副標 hashtag UI 修正
   - 修正副標 UI 會換行導致不美觀的問題
   - 原本的標籤區塊改成橫向可捲動的單行容器，捲軸本身是隱藏的（no-scrollbar），介面乾淨

3. 使用者自訂 hashtag
   - 在每張角色卡片的標籤行最右邊，多了一個「+ 標籤」按鈕
   - 點擊後展開一個 inline 小輸入框，輸入後按 Enter 或 ✓ 即可新增，Esc 取消
   - 每個自訂標籤顯示為帶 # 的玫紅色 pill，有 × 按鈕可以移除
   - 標籤會直接儲存到 IndexedDB，重整後不會遺失
   - Mock 角色也預設加入了範例標籤（主角、女主角、獵人等）供參考
