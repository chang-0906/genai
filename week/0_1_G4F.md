<div align="center">

# G4F

</div>

> [Github](https://github.com/xtekky/gpt4free/tree/main)
>
> 一個「整合多個 AI 來源的工具」，讓你用類似 OpenAI API 的方式呼叫各種模型（甚至 GPT-4）
> 
> 查看可用模型：[G4F-working](https://github.com/Free-AI-Things/g4f-working/blob/main/working/models.txt)（每日更新）

👉 它不是模型
👉 也不是官方 API

👉 它是「中間層 / aggregator」

> [!WARNING]
> 可以在 Python 中使用，但是不能用在 Cline、Dify

# 1️⃣ ⚙️ 它怎麼運作（核心重點）

## 🧩 1. 整合很多 provider

其實只是「模仿 OpenAI API 介面」

---

## 🧨 2. 關鍵：它有些 provider 是「非官方的」

這才是重點👇 很多 provider 是這樣來的：

### 方法 A：抓網站 API（reverse engineering）

* 模擬網站（像 Poe / You.com）
* 偷用它們後端的 API

👉 假裝自己是某個網站的前端

---

### 方法 B：借別人的付費帳號流量

有些情況是：

> 把 request 偽裝成「某個已付費網站的請求」

👉 結果：

* 你免費用 GPT-4
* 但**帳單是別人付**

---

### 方法 C：browser automation / cookie

repo 也提到：

* 需要 cookies
* headless browser ([GitHub][1])

👉 意思是：

> 直接模擬真人在用網站

---

# 2️⃣ 💸 為什麼可以「免費」？

👉 核心答案：

> **因為不是你在付錢**

---

更精確說：

### ✔️ 合法部分（少數）

* 接開源模型（例如本地 LLM）
* 接真正免費 API

---

### ⚠️ 灰色 / 爭議部分（主要）

* 利用別人網站的 API
* 繞過付費限制 ([MarkTechPost][3])

---

👉 所以才會出現：

* OpenAI 警告 / 法律威脅 ([PC Gamer][4])

---

# 3️⃣ ⚠️ 風險（非常重要）

我直接講現實面👇

## 🚨 1. 法律風險

* 可能違反服務條款（ToS） ([apidog][5])
* 有些情境可能接近「未授權使用」

---

## 🔐 2. 資安風險（更嚴重）

👉 你的 prompt 可能會：

* 經過不明 server
* 被記錄
* 被第三方看到 ([skywork.ai][6])

👉 結論：

> ❌ 不要丟任何敏感資料

---

## ⚡ 3. 穩定性很差

* provider 常壞
* API 隨時被封
* 回應不穩定 ([skywork.ai][6])

---

# 4️⃣ 🧰 它有哪些功能？

其實功能還滿強的（技術上）：

---

## 🔹 1. OpenAI-compatible API

```python
g4f.ChatCompletion.create(...)
```

👉 可以直接當替代 OpenAI SDK

---

## 🔹 2. 多模型切換

* GPT-4
* Claude
* Gemini
* DeepSeek 等

👉 一個 interface 用全部

---

## 🔹 3. Local model 支援

👉 可以接：

* 本地 LLM
* Ollama / llama.cpp

---

## 🔹 4. 多媒體（部分 provider）

* 生成圖片
* 音訊
* 影片 ([GitHub][1])

---

## 🔹 5. 負載平衡 / fallback

* 換 provider
* 自動 retry ([apidog][5])

👉 這點其實設計不錯

---

## ❌ 不建議

* 商業產品
* 正式專案
* 有敏感資料

---

# 6️⃣ 最小可行方案（於 modal notebook）

## 01 Gemini

> 可以先透過連結 [DOC]() 中的 base_url 看看不同 url 有支持哪些 model

```python
from openai import OpenAI

client = OpenAI(
    api_key="你的實際 API Key",
    base_url="https://g4f.space/api/gemini",
)

response = client.chat.completions.create(
    model="models/gemini-2.5-flash",
    messages=[
        {"role": "user", "content": "請用繁體中文簡短介紹你自己。"}
    ],
    temperature=0.7,
)

print(response.choices[0].message.content)
```
